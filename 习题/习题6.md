# 1.写出下列程序的输出结果。
```python
# 1.py
class Student():
    def __init__(self):
        self.name = "张三"
        self.subject = "英语"
    def study(self):
        print(self.name ,"正在学习",self.subject,"这门课程")
student1 = Student()
student1.study()
```
答案：张三 正在学习 英语 这门课程
